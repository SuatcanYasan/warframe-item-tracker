# Kolay Kurulum (Teknik Bilgi Gerektirmez)

Bu proje yerel bilgisayarinda tek tikla calisabilir.

## 1) Node.js kur

- https://nodejs.org adresinden **LTS** surumunu indir ve kur.
- Kurulumdan sonra bilgisayari bir kez yeniden baslatman iyi olur.

## 2) Projeyi ac

- Bu klasoru indir ve ac.
- `KULLAN.bat` dosyasina cift tikla.
- Masaustu pencere uygulamasi istersen `MASAUSTU.bat` dosyasina cift tikla.

Bu script su islemleri otomatik yapar:

- Gerekliyse bagimliliklari kurar
- Gerekliyse arayuzu build eder
- Uygulamayi baslatir ve tarayiciyi acar

## 3) Uygulamayi kullan

- Tarayicida `http://localhost:3000` acilir.
- Kapatmak icin scriptin acik oldugu terminal penceresinde `Ctrl+C` yap.
- Ilk acilista kucuk bir sihirbaz acilir; dil ve temayi secip `Tamamla` diyebilirsin.

## Not

- Gelistirici modu icin terminalde `npm run dev` kullanabilirsin.
- Tek komutla build+calistirma icin `npm run easy` kullanabilirsin.
